# MicroHash

Tiny file hashing utility.

## Usage

python3 -m microhash <file>

Algorithms:

- MD5
- SHA1
- SHA256
- SHA512

Designed for learning, file integrity verification, and defensive security workflows.

---

## Overview

A minimal utility for calculating and comparing cryptographic hashes.
